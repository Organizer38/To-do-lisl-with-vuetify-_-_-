# Список дел (Todo App)

Простое приложение «Список дел» c компонентами Vuetify.

# Запуск

Просто откройте файл `index.html` в браузере — никаких зависимостей устанавливать не нужно.

# Возможности

- Добавление задач (с клавиатуры или кнопкой)
- Отметка выполнения кликом по задаче
- Пометка задачи как «Важная»
- Категории: Работа, Личное, Покупки
- Фильтры: Все / Активные / Выполненные / Важные
- Удаление отдельных задач
- Очистка всех выполненных задач
- Счётчик прогресса

# Структура проекта

```
To-do list/

└── main.css     # Стили
```
